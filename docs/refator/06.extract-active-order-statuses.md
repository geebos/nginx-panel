# 06. Extract active ACME order statuses constant

## 发现的问题

in-flight 订单状态列表在 renewal（2 处）与 scheduler（2 处）硬编码重复。

## 证据

相同数组：`preparing, waiting_http, waiting_dns, validating, validated, downloading`

- `renewal.ts` active-order gate（查询 + tx 内）
- `scheduler.ts` due poll + shutdown persist
- cleanup 查询使用 terminal 列表（改为共享 `terminalOrderStatuses`）

## 方案

在 `order-status.ts` 增加 `activeOrderStatuses`；调用点 import。不改 create 路径的 `notInArray(terminal)` 语义。

## Design Challenger

APPROVE（本轮仅 renewal 也可；扩展到 scheduler 同列表属同概念、同常量）。

## Code Reviewer

**APPROVED** — 无 BLOCKING。

## 验证

- order-status + renewal + scheduler → 15 pass
- typecheck / lint clean
