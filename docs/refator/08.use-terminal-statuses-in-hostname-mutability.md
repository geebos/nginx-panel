# 08. Use terminalOrderStatuses in hostname mutability gate

## 发现的问题

Iteration 5 抽取 `terminalOrderStatuses` 时遗漏
`assertHostnamesMutable`（`domain/validation.ts`）仍硬编码终态列表。

## 证据

```ts
notInArray(acmeOrders.status, ["succeeded", "failed", "expired", "cancelled"])
```

与 create-order / manager 的 active-order 门禁同语义（非终态订单锁定 hostname）。

## 方案

Import `terminalOrderStatuses` 并替换字面量。不改 `notInArray(terminal)` 为 `inArray(active)`（未知状态保守锁定行为保持）。

## 范围外

前端 `ssl.tsx` / manager ssl UI 的 `terminalOrders` 无法 import worker 模块；另案（shared 包）再议。
