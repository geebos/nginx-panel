# 15. Extract parseManagerSnapshot module

## 问题

`parseManagerSnapshot` 已在 `manager/service.ts` 定义，但 5 处 production 仍 inline `managerConfigSchema.parse(JSON.parse(...))`。且 `service` 依赖 `domain/validation`，若 validation 反向 import service 会形成循环。

## 方案

- 新建 `src/worker/lib/manager/snapshot.ts` 放置纯函数（对称 `domain/snapshot.ts`）
- `manager/service.ts` re-export 保持 `manager/certificate` 等旧 import 兼容
- 替换 root-input / activation / validation / runner / config-test
- 不改 auth `safeParse`
