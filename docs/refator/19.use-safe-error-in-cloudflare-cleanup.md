# 19. Use safeErrorMessage in cloudflare-cleanup

## 问题

`cloudflare-cleanup` 本地 `safeError` 仅 slice(500)，失败写入 `acme_orders.errorMessage` 时不脱敏 URL；同域其它路径已用 `safeErrorMessage`。

## 方案

删除本地 helper，改 `safeErrorMessage(error, "Cloudflare DNS cleanup failed")`（默认 `[URL]` 占位）。行为变更：URL 脱敏。
