# 25. Extract sslConfigStatus

## 发现的问题

域名 SSL 配置 badge 策略（`certificateId → active` / `enabled → pending` / else `disabled`）在 FE overview、FE ssl 页与 worker 域名列表 API 三处重复。记录 23 已证明语义修正会漏改。

## 问题证据和影响范围

- `src/pages/[locale]/domains/overview.tsx`
- `src/pages/[locale]/domains/ssl.tsx`
- `src/worker/routes/domains.ts`（`sslStatus` 字段）

## 原始设计

三处内联同一三元表达式。

## 最小修改方案

在 `src/shared/schemas/domain.ts`（`sslConfigSchema` 旁）导出：

- `SslConfigStatus`
- `sslConfigStatus({ certificateId?, enabled })`

接线三处；`ssl.tsx` 保留 `config` 缺失 → `"disabled"`。

**APPROVE**
