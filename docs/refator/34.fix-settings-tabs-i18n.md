# 34. Fix settings tabs and general breadcrumbs i18n

## 发现的问题

- `SettingsTabs` 全部英文硬编码标签 + aria-label
- `settings/general` breadcrumbs 硬编码 `Settings` / `General`，与其它 settings 页不一致

## 问题证据和影响范围

- `src/components/pages/settings/tabs.tsx`
- `src/pages/[locale]/settings/general.tsx`
- 已有 `common:settings.*.breadcrumb` 与 `common:settings.breadcrumbs.settings`

## 原始设计

tabs 数组带英文 `label`；general 用字面量 breadcrumbs。

## 最小修改方案

- tabs 改 slug 列表 + `t(\`common:settings.${slug}.breadcrumb\`)`（对齐 DomainTabs）
- 新增 `common:settings.tabs.ariaLabel`、`common:settings.general.breadcrumb`
- general breadcrumbs 用 i18n key；Settings 父级 href 对齐其它页 `/settings/nginx`
