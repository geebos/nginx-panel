# 35. Fix diagnostics zh-CN copy still equal to English

## 发现的问题

Settings → Diagnostics 多处 `zh-CN` 与 `en` 相同，中文界面出现英文残片（标题、面包屑、Active Domain 标签等）。

## 问题证据和影响范围

- `public/locales/common.json` → `settings.diagnostics.*`
- 页面仅用 `t("common:settings.diagnostics...")`，无代码逻辑变更

## 原始设计

部分 key 只填了英文，zh-CN 原样复制。

## 最小修改方案

仅更新 diagnostics 下用户可见且应中文化的 `zh-CN` 文案；保留 SAN / SHA-256 等技术专名。
