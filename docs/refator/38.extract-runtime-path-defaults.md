# 38. Extract nginxRuntimeRoot / certificateDataRoot

## 发现的问题

多处硬编码 `NGINX_RUNTIME_ROOT || "/data/nginx"` 与 `CERTIFICATE_DATA_ROOT || "/data/certs"`，默认路径漂移风险。

## 问题证据和影响范围

- runner / diagnostics / storage / verifier / rotator / settings route
- certificate-store 私有 `certificateRoot()`

## 最小修改方案

`src/worker/lib/runtime/paths.ts`：

- `DEFAULT_*` 常量
- `nginxRuntimeRoot(override?)` / `certificateDataRoot(override?)`（`||` 语义）

接线上述 call sites；HTTPS catch-all 硬编码路径不动（既有）。

## Design Challenger 结论

**APPROVE**（薄 getter SSOT；不扩 env 注册表）

## TDD 过程

paths.test.ts：默认 / env / override / 空串回落

## 最终修改

- `src/worker/lib/runtime/paths.ts` + `.test.ts`
- runner, diagnostics, storage, verifier, rotator, settings, certificate-store
